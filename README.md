# Linked List in C

Simple linked list implementation in C (using only the C standard library).

## Example Usage

See `src/main.c`

## Prerequisites

- doxygen
- ruby
- ceedling
- gcovr

## Build

### Library & Driver

```bash
make
```

### Tests

```bash
make test
```

## Run Example

```bash
make run
```

## Run Tests

This assumes you have ruby/ceedling

```bash
make test
```

### Documentation

```bash
make doc
```
